# Data Insights

This is a beginner-level data analytics project created to practice Git, GitHub, and basic data analysis concepts.

The goal of this project is to gradually build skills in analyzing small datasets and documenting insights in a structured way.

- Basic data analysis (learning stage)

## SQL Data Analysis

Created a sales database and table using MySQL.  
Inserted sample sales data and performed analysis to calculate total revenue, revenue by category, and top-selling products.

### Skills Practiced
- SQL DDL & DML
- Aggregate functions
- GROUP BY, ORDER BY

### Advanced SQL Window Functions
- Used PARTITION BY for department-wise ranking
- Compared ROW_NUMBER, RANK, and DENSE_RANK
- Identified top-paid employees per department using window functions
